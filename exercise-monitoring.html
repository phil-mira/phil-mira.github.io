Exercise Monitoring with Inertial Sensors

A project focused on using machine learning to monitor exercise activities through wearable sensors.

Project Overview

This project explores the use of machine learning algorithms to analyze data from a network of wearable inertial sensors. The goal was to accurately monitor and classify various exercise activities, providing real-time feedback to users.

Key Contributions

- Developed a machine learning pipeline for processing sensor data.
- Designed and implemented a feature extraction method to improve classification accuracy.
- Conducted extensive testing and validation using real-world datasets.

Technologies Used

- Python, TensorFlow, and Scikit-learn for machine learning.
- MATLAB for signal processing.
- Wearable sensor hardware for data collection.
