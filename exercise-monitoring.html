Machine Learning for Exercise Monitoring from a Network of Inertial Sensors

Project Overview

This project was done in collaboration with a small startup in the UK with the aim of developing algorthims to complement their wearable sensors which aimed to be used as an exercise monitoring device. The goal of the prokect was develop the algorithms able to classify the exercise being performed by the wearer, count the number of reps and determine the quality in terms of technique of the exercises being performed.
